PayG Payment Extension for Opencart
This extension provides non seamless integration with OpenCart, allowing payments for merchants from their Customer.

Installation
Copy all files/folders recursively to opencart installation directory.

Go to Admin Panel, Extensions->Payments and install the PayG gateway extension.

Click on Edit next to PayG and do the following:

Add your MerchantKey Id
Add your Authentication Key
Add your Authentication Token
Add Your SecureHashKey
Change plugin status to Enabled
Save the plugin settings
